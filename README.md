matsuri3
========

第四回目Matsuriサイト
12/15開催！！

* Usage

```bash
# Please Fork
 $ git clone https://github.com/yourname/matsuri4.git
 $ cd matsuri4
 $ git checkout -b [your_branch_name]
 $ bundle install
 $ bundle exec middleman
```

* Release
```bash
 $ rake build    # Compile all files into the build directory
 $ rake publish  # Build and publish to Github Pages
```
